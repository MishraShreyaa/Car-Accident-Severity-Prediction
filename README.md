# Car Accident Severity Prediction Using Machine Learning

## Overview

This project predicts accident severity using weather, location, traffic, and environmental conditions from the US Accidents dataset.

## Dataset

- Source: US Accidents Dataset (Kaggle)
- Records Used: 300,000+
- Features: Weather, Traffic Signals, Location, Time Features, Road Conditions

## Project Workflow

1. Data Cleaning
2. Missing Value Treatment
3. Exploratory Data Analysis (EDA)
4. Feature Engineering
5. Outlier Detection and Treatment
6. One-Hot Encoding
7. Model Training
8. Model Evaluation
9. Feature Importance Analysis

## Models Used

- Decision Tree
- Random Forest
- XGBoost

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Confusion Matrix
- ROC Curve
